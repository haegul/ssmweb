Matrix 변환- Scale

Scale함수는 좌표계 x, y, z축의 눈금이 전역 좌표계 눈금의 sx, sy, sz 배가 되도록 크기를 조잘합니다.

예시

  1. Scale(3,3,3);
    // X,Y,Z축의 방향으로 3배만큼 크기를 조절합니다.
  2. Traslate(1,0,0);
    // x+축의 방향으로 1만큼 원점(0,0,0)을 옮깁니다. -> 원점의 좌표(1,0,0)*3
  3. drawBox(1,0,0,0);
    // 원점(3,0,0)에 box를 그립니다.

Notice

Translate와 Rotate의 순서에 주의합니다. Translate를 먼저할 경우 이동한다음 원점의 좌표를 회전시키는 반면, Rotate를 먼저할 경우 회전 시킨후 이동하기 때문에 결과가 달라집니다.
