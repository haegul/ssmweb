지엘은 단순히 하나의 행렬만 제공하는 것이 아니라 행렬 스택을 제공합니다. 행렬에 대한 Push/Pop 작업은 실제로 모델 좌표계의 Push/Pop을 의미합니다. 즉 행렬을 저장하고 복원한다는 것은 이전 좌표계의 원점 위치, 축 박향, 축눈금 길이등을 저장하고 복원하는 것에 해당합니다.

예시

  1. Translate(3,0,0)
    // 원점의 좌표를 (3,0,0) 이동시킵니다. - > 원점의 좌표(3,0,0)
  2. PushMatrix();
    // 행렬 스택에 현재 원점의 좌표를 저장합니다. -> 행렬 스택 Top(3,0,0)
  3. Translate(3,0,0)
    // 원점의 좌표를 (3,0,0) 이동시킵니다. - > 원점의 좌표(6,0,0)
  4. Translate(0,5,0)
    // 원점의 좌표를 (0,5,0) 이동시킵니다. - > 원점의 좌표(6,5,0)
  5. PopMatrix();
    // 행렬 스택에서 이전 원점의 좌표를 pop하여 복원시킵니다 -> 원점의 좌표(3,0,0)
