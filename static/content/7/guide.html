원구 그리기 - High API

지엘에서는 정점을 일일이 찍어서 오브젝트를 만드는 방법도 있지만 사용자의 편의를 위해 모델링 된 몇가지 물체가 존재합니다.(GLUT에서 제공)

적육면체(Cube), 원구(Sphere), 원환체(Torus), 원뿔(Cone), 정20면체(Lcosahedron) 등등이 있습니다.

지엘 라이브러리마다 제공되는 함수와 파라미터가 다르며 대표적인 원구를 그리는 함수는 다음과 같습니다.

OpenGL : glutSolidSphere(GLdouble radius, GLint slices, GLint stacks);

Processing : sphere(float r); shpereDetail(ures, vres);
